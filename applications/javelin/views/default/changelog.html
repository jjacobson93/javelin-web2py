v1.0.0 and below

Overview

Provides base functionality for management and organization of the community.

Features

  • A full set of rich controllers:
    • People: Access and manage information about the people in the community.
    • Groups: Organize people into groups.
    • Events: Manage and create events.
    • Messages: Send SMS messages to people and groups.
    • Admin: An admin interface for customization and approving users.
  • Specialized controllers for Crusher Crew:
    • Orientation: The interface for Crusher Crew Orientation
    • Peer Support: Create and monitor Peer Support.
    • Scores: The interface for scoring crews during training.
